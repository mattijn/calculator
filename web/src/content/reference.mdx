# Language Reference

v1 browser evaluator specification

## Tokens

- Numbers: integer or decimal (`2`, `1.03`)
- Grouping: `(` and `)`
- Binary operators: `+`, `-`, `脳`/`x`/`*`, `梅`/`/`, `鈫慲, `鈫揱, `鈬揱
- Unary operators: prefix `+`, `-`, `梅`/`/` (reciprocal)

## Precedence And Associativity

1. Unary operators (right-associative chaining)
2. `鈫慲, `鈫揱, `鈬揱 (left-associative)
3. `+`, `-`, `脳`, `梅` (left-associative)

This ordering mirrors the legacy implementation behavior for compatibility, while adding explicit unary `梅` support.

## Semantics

- `a鈫慴` = `a ** b`
- `a鈫揵` = `a ** (1 / b)`
- `a鈬揵` = `log(a) / log(b)`
- `梅x` = `1 / x`

## Error Behavior

- Parse failures report invalid syntax for unexpected or missing tokens.
- Non-finite results (for example division by zero) are rejected.
